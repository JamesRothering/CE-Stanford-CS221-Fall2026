# For the hiring manager

If you are reading this, you’re likely hiring for a role that values learning, documentation, and clear thinking.

This repo shows:

- How I turn lecture notes into working code.
- How I explicitly capture questions, goals, and tasks.
- How I document my mistakes and fixes.
- How I structure a personal project like a small open‑source project.

If you want to see more, I can walk through any assignment and explain:

- Why I wrote it that way.
- What I would do differently now.
